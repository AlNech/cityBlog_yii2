## Запуск проекта:

| Шаг | Команда               |
|----|------------------------|
| 1  | `make init`            |
| 2  | `make start`           |
| 3  | `make migrate up`      |

### Команды Make

| Команда                 | Описание                                                        |
|-------------------------|-----------------------------------------------------------------|
| `make init`             | Полная установка проекта с нуля                                 |
| `make start`            | Запускает Docker-контейнеры                                     |
| `make end`              | Удаляет Docker-контейнеры                                       |
| `make migrate up`       | Применение миграции                                             |
| `make migrate down`     | Отмена миграции                                                 |
